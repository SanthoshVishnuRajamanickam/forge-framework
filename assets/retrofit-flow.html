FORGE Retrofit

Reverse-engineer project state from git history + codebase analysis

2 Modes

Standard (default)

~28K tokens
90 seconds | 4 agents (sonnet)
PROJECT.md + ROADMAP.md + STATE.md + codebase map (7 docs)

Full

~50K tokens
3 minutes | 5 agents (4 sonnet + 1 haiku)
+ MILESTONES.md + phase reconstruction + plan archaeology

Retrofit Flow

Confidence Scoring

HIGH — from tags + CHANGELOG (human-curated)
MEDIUM — from conventional commits + file patterns
LOW — from time clustering only (needs user review)

Agent Architecture

Agent 1: Stack

sonnet
Tech stack, frameworks, dependencies, build tools

Agent 2: Architecture

sonnet
Directory structure, patterns, component relationships

Agent 3: Conventions

sonnet
Code style, naming, testing patterns

Agent 4: Concerns

sonnet
Tech debt, security issues, performance risks

Agent 5: Git History

haiku
Milestones, phases, features, decisions from git + CHANGELOG. Cost-optimized model.
