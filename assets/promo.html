Example: Building a Web App

  • Milestone
    v1.0 — MVP Launch 3 phases, targeting March release
  • Phase 1
    Authentication System 2 plans: user registration + JWT management
  • Plan 01
    User Registration PLAN → APPLY (3 tasks, unit tests per-task) → VERIFY (E2E) → UNIFY
  • Plan 02
    JWT Token Management PLAN → APPLY (4 tasks, TDD mode) → VERIFY (MCP-driven) → UNIFY
  • Phase 2
    API Layer 3 plans: CRUD endpoints + validation + error handling
  • Phase 3
    UI Components 2 plans: design system + page layouts
  • Complete
    v1.0 shipped All phases verified, milestone archived, git tagged, release notes generated
