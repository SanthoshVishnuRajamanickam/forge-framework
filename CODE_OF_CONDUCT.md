# Code of Conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## The short version

- Be respectful. Disagree with ideas, not people.
- Assume good faith. Most misunderstandings are not malice.
- No harassment, discrimination, or personal attacks.
- Keep discussion focused on the project.

## Reporting

If you experience or witness behavior that violates this code, open a private
report to the maintainer via the email listed on the GitHub profile for
`santhoshvishnu`. Reports are handled confidentially.

## Scope

This code of conduct applies to all project spaces — issues, pull requests,
discussions, and any other forum associated with forge-framework — and when
an individual is representing the project in public.
